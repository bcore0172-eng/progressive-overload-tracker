# Progressive Overload Tracker

A simple web-based progressive overload tracker built with HTML, CSS, and JavaScript.

## Features
- Track exercises, sets, reps, and weight
- Clean, simple UI
- No backend required (runs fully in the browser)

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Run
Just open `index.html` in your browser.

## Future Improvements
- Local storage support
- Mobile-first layout
- Charts and progress visualization
